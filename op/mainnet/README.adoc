= OP Mainnet Node

Full-node setup, using **op-geth** and **op-node**.

NOTE: Switched from op-reth to op-geth due to unreliable `eth_getProof` support. For the previous reth-based setup, see commit `6e04c5e6`.

== Steps

. Copy and configure environment:
+
[source,sh]
----
cp .env.example .env
# Edit .env with your L1 RPC endpoints if needed
----

. Generate JWT token:
+
[source,sh]
----
./generate-jwt-token.sh
----

. Run containers:
+
[source,sh]
----
docker compose up --detach
----

. Check sync status:
+
[source,sh]
----
curl -s http://localhost:23011 -X POST -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"optimism_syncStatus","params":[],"id":1}' | jq
----
