= Atlas - RPC Infrastructure

Container-based node setups for EVM networks.

== Supported Chains

[cols="1,1,1"]
|===
| Chain | Networks | Type

| Ethereum | mainnet, sepolia | full-node
| Linea | mainnet, sepolia | full-node
| Base | mainnet, sepolia | full-node
| OP | mainnet, sepolia | full-node
| OP Commitment | mainnet, sepolia | gossip-only
|===

=== Port Pattern

Format: `2CNLP`

- **2** = Prefix
- **C** = Chain (0=Ethereum, 1=Linea, 2=Base, 3=OP, 4=OP Gossip)
- **N** = Network (0=mainnet, 1=sepolia)
- **L** = Layer (0=EL, 1=CL)
- **P** = Port (0=P2P, 1=RPC-HTTP, 2=RPC-WS)
